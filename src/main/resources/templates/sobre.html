Emotion


Sistema que analisa a imagem de um rosto humano e reconhece as emoções expressadas, para isso faça o upload de uma imagem e ela será analisada por um modelo de Inteligência Artificial treinado para reconhecer as 7 emoções universais básicas, definidas por Paul Ekman.

Projeto desenvolvido como Trabalho de Conclusão de Curso, para o curso de Tecnologia em Análise e Desenvolvimento de Sistemas, utilizando a linguagem Java com o framework Spring para desenvolver a interface do usuário e utilizado a linguagem Python para desenvolver a inteligência artificial, criada com uso do Framework TensorFlow.

Orientador: Cristiano Agosti
